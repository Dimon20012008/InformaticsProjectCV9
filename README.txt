Name of project: "Krivoy Chertila"

Description: it is a game, where you need to draw a contour in the air with your finger

Installation and launch: make sure to have all libs from requirements.txt, start main.py

Usage: make sure, that your index finger is on contour and there's a blue circle, which detects your finger.
Press any key on keyboard and follow the contour. When you return to the start (yellow circle) a score appears.

Completness of project: want to add more contours and their shuffle, color of your path changes, as you get farther from contour.
